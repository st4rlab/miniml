Installation

Install Github-Uikit's source and JavaScript files via clone, fork, or npm. Package managed installs donâ€™t include documentation or our full build scripts. You can also use our npm template repo to quickly generate a Bootstrap project via npm.

The easiest way to install quickly is to clone the GitHub repository.

git clone https://github.com/isaacjosephhorton/github-uikit.git

Make sure you have jekyll installed and you can run a build test locally with:

bundle exec jekyll serve

Or install it yourself as:

$ gem install github-uikit
