Interface building blocks for professional website interaction.

Installation

Install Github-Uikit's source and JavaScript files via clone, fork, or npm. Package managed installs donâ€™t include documentation or our full build scripts. You can also use our npm template repo to quickly generate a Bootstrap project via npm.

Add this line to your Jekyll site's Gemfile:

gem "github-uikit"

And add this line to your Jekyll site's _config.yml:

theme: github-uikit

And then execute:

$ bundle

Or install it yourself as:

$ gem install github-uikit

Features

Built With UiKit

A lightweight and modular front-end framework for developing fast and powerful web interfaces.

Tags and Archive page Included

Tags will automatically fill the tag cloud and archive page when properly listed in front matter.

Algolia Search Ready

Use Algolia's instant search! It's insanely fast and pretty easy to set up and configure.
